# 腾冲幺子分 - 微信游戏小程序

## 项目概述
腾冲幺子分是一款经典扑克牌游戏，使用54张牌，四人两组对战。本小程序实现了完整的游戏逻辑和 AI 对手。

## 核心规则实现

### 牌组
- 54张：红桃/黑桃/梅花/方块各13张 + 大小鬼
- 大小顺序：大鬼 > 小鬼 > A > K > Q > J > 10 > ... > 2
- 分值牌：5(5分) / 10(10分) / K(10分)

### 游戏流程
1. **发牌** → 每人12张 + 6张底牌，逆时针轮换
2. **叫分** → 75-100分，5的倍数，必须超过上家
3. **选主牌** → 庄家选定花色，大小鬼永远为主牌
4. **扑底牌** → 庄家用6张手牌换底牌（不能扑分牌）
5. **加倍** → 吃牌方可加：倒(2x)/擂(3x)/窝(4x)/抓(5x)
6. **出牌** → 12轮，大吃小，主杀副，无牌可垫
7. **结算** → 打牌方吃够叫分赢，光头3杯底酒

### AI 策略
- **叫分**：评估手牌强度（大小鬼/大牌/分牌权重）
- **出牌**：判断对家是否在赢 → 决定是否抢牌
- **底牌**：保留分牌和主牌大牌，打出非分牌小牌
- **难度**：简单/普通/困难三级
